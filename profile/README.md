## Quick Start with CrystalMark Download

1. **Get CrystalMark Download** — Download the portable `.zip` (no installation required) or the installer version.
2. **Extract and launch** — Unpack the archive and double-click `CrystalMark.exe`. The main window opens in under one second.
3. **Select your test categories** — By default, **CrystalMark Download** tests CPU, hard disk, graphics, and memory. Uncheck any category you want to skip.
4. **Run the full benchmark** — Click the “Benchmark” button. **CrystalMark Download** will run all selected tests in sequence.
5. **View your results** — A total score appears instantly. **CrystalMark Download** also shows individual subscores for CPU (ALU/FPU), HDD (sequential/random), GDI, OpenGL, and memory.
6. **Compare and save** — Export your results as a text file, CSV, or JSON. **CrystalMark Download** allows you to compare against previous runs.
7. **Loop for stability testing** — Enable loop mode in **CrystalMark Download** to run all tests continuously for 1 hour, 24 hours, or indefinitely.
8. **Work normally** — Close **CrystalMark Download** when finished. It leaves no background processes, no startup entries, and no telemetry.

## Understanding CrystalMark Download Core Components

**CrystalMark Download** is not just a simple benchmark. It includes several precisely engineered test modules:

- **CPU ALU (Arithmetic Logic Unit)** — Measures integer calculations, logic operations, and bit fiddling speed.
- **CPU FPU (Floating Point Unit)** — Tests single‑precision and double‑precision math, including sine, cosine, square root, and division.
- **Hard Disk Drive benchmark** — Sequential read/write, random 4K read/write, and access time measurement.
- **Memory benchmark** — Bandwidth and latency tests for RAM (DDR3, DDR4, DDR5, and LPDDR).
- **Graphics (GDI)** — Standard Windows 2D drawing performance: lines, rectangles, bitmaps, and fonts.
- **Graphics (OpenGL)** — 3D rendering performance via the OpenGL API using basic shapes and textures.
- **Overall score calculation** — **CrystalMark Download** combines all sub‑scores into one easily comparable number.

All these modules work together seamlessly in **CrystalMark Download**. You never need to install separate testing tools or guess where a performance bottleneck lies.
